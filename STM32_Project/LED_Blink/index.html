💡 LED Blink – STM32F401RE

A minimal **bare-metal firmware project** that toggles the onboard LED (PA5) on the STM32F401RE Nucleo board. This is the foundation for embedded firmware testing, clock configuration, and GPIO initialization.

Core Contribution: System clock setup, GPIO configuration, and firmware loop for periodic toggling.

🔑 Key Highlights

  • HAL-based firmware for STM32F401RE
  • Configured system clock using PLL (84 MHz)
  • Blinked onboard LED (PA5) with 500 ms interval
  • Lightweight project template for further STM32 projects

⚙️ Hardware Setup

  • MCU: STM32F401RE (Nucleo-64)
  • Onboard LED: PA5 (LD2)

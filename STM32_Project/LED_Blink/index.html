💡 LED Blink – STM32F401RE

The **LED Blink** project is the classic "Hello World" of embedded systems. It toggles the onboard LED (PA5) every 500ms using the STM32 HAL library.

⚙️ Features

  • Runs on STM32F401RE (Nucleo board)
  • Uses HAL GPIO functions
  • Simple 500ms toggle delay

🔌 Connections

Onboard LED (LD2) is already connected to pin PA5. No extra wiring needed.

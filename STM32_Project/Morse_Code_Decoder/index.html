⚡ Morse Code Decoder – STM32F401RE

A real-time Morse Code Decoder implemented on STM32F401RE. It captures button input, decodes dot/dash sequences into characters, and provides feedback through an LCD and EEPROM for persistent storage.

Core Contribution: Designed timing logic for Morse input, integrated I²C LCD & EEPROM, and validated end-to-end system.

🔑 Key Highlights

  • Dot/Dash input captured via push button with precise timing
  • Live decoded characters displayed on 16x2 I²C LCD
  • Persistent message storage on EEPROM 24C04
  • Onboard LED for input feedback

⚙️ Hardware Setup

  • MCU: STM32F401RE (Nucleo-64)
  • Display: 16x2 LCD via I²C (PB8 → SCL, PB9 → SDA)
  • Storage: EEPROM 24C04 (I²C)
  • Input: Push Button (PC13)
  • Feedback: Onboard LED (PA5)

⚡ Morse Code Decoder – STM32F401RE

This project demonstrates a **real-time Morse Code Decoder** designed and implemented on the STM32F401RE platform. The system captures input via a tactile push button, processes the Morse signals, decodes them into characters, and outputs results both on an **I²C-driven LCD display** and **non-volatile EEPROM storage**.

Core Contribution: Firmware design, real-time input processing, peripheral integration (I²C, GPIO), and end-to-end system validation.

🔑 Key Highlights

  • Precise timing logic for dot/dash recognition and spacing
  • Integrated EEPROM storage (24C04) for persistent decoded messages
  • Real-time LCD updates for immediate feedback
  • Minimalistic HAL-based architecture for STM32F4 series

⚙️ Hardware Setup

  • MCU: STM32F401RE (Nucleo-64)
  • Display: 16x2 LCD over I²C (PB8 → SCL, PB9 → SDA)
  • Storage: EEPROM 24C04 on shared I²C bus
  • Input: Push Button (PC13)
  • Feedback: Onboard LED (PA5)
